# SpeedLink
**SpeedLink** — это быстрый и безопасный VPN-клиент на базе ядра Sing-box.

## Особенности
- Поддержка протоколов VLESS, VMess, Reality, TUIC, Hysteria.
- Автоматический выбор самого быстрого сервера.
- Режим TUN для проксирования всего трафика.
- Полностью открытый исходный код.
